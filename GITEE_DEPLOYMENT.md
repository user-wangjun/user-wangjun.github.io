---

### 步骤 1: 进入 Pages 服务管理页面

**点击以下链接（按 Ctrl+ 点击打开）**：
```
https://gitee.com/WangShangYiJun/smart-calendar/pages
```

或手动操作：
1. 打开仓库：https://gitee.com/WangShangYiJun/smart-calendar
2. 点击顶部菜单「**管理**」
3. 左侧菜单选择「**Pages 服务**」

---

### 步骤 2: 配置并启用 Pages

在 Pages 服务页面：

1. **来源分支**：选择 `main`
2. **发布目录**：填写 `/`（根目录）
3. 点击「**保存**」按钮

---

### 步骤 3: 等待审核和构建

- **首次使用**：需要审核（1-2 小时）
- **审核通过后**：自动构建（1-3 分钟）
- **构建完成后**：显示访问地址

---

### 步骤 4: 获取访问地址

构建成功后，你会看到：

```
✅ Pages 服务已启用

https://WangShangYiJun.gitee.io/smart-calendar/
```

**点击即可访问你的应用！**


## 🐛 常见问题

### Q1: 页面显示 404？

**A**: 
- 等待 2-3 分钟，构建需要时间
- 检查 Pages 服务是否已启用
- 确认 base 路径是否正确

### Q2: 样式错乱/资源加载失败？

**A**:
- 清除浏览器缓存
- 检查 vite.config.js 中的 base 路径
- 确保是 `/smart-calendar/`

### Q3: 如何更新代码？

**A**:
```bash
git add .
git commit -m "feat: 更新功能"
git push gitee main
```
Gitee Pages 会自动重新构建。
